What are the different ways to select an element in the DOM?

There are different ways to select elements in DOM.These are getElementsByTagName,getElementsByClass, getElementByID,getElementByQuerySelector, QuerySelectorAll, getAttribute, setAttribute, innerText, innerHTML etc.

What is the difference between innerHTML, innerText, and textContent ?

innerHTML returns the HTML contents including the tag. innerText and textContent returns the text inside the tag, not returns the tag. They both does same ut there is a key different between them. InnerText returns text with css styling. But textContent ignores CSS styling.

What is event delegation in the DOM?

Attaching a single eventListener to parent element instead of adding multiple eventListener to child elements.

What is event bubbling in the DOM?

Event bubbling is a concept by which user gives command to browser to get the target element. When user clicks on an element it keeps bubbling till root and get the target.
